phyloHiC
========

Make a phylogeny using the chromatin structure information extracted from
Hi-C experiments.

Requirements
------------

* Python 3 (at least 3.6 for the [f-string][1] notation)
* [PyToolz][2]
* [tqdm][3] (optional, used to display progress bar)
